# Bank Loan Dashboard

## Project Overview
This project analyzes bank loan data using SQL and Power BI to explore portfolio performance, lending trends, and borrower segmentation. SQL was used to query and structure the dataset, while Power BI was used to build an interactive dashboard for visualization and analysis.

## Tools Used
- SQL
- Power BI

## Dashboard Structure
The dashboard consists of 3 pages:
1. **Summary** — key portfolio metrics such as total loan applications, funded amount, amount received, average interest rate, average debt-to-income ratio, and good vs bad loan composition.
2. **Overview** — breakdown analysis by month, state, loan term, employment length, loan purpose, and home ownership.
3. **Details** — loan-level records for deeper exploration and filtering.

## Key Metrics
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average Debt-to-Income Ratio
- Good Loan Percentage
- Bad Loan Percentage

## Key Insights
- Good and bad loan composition helps evaluate portfolio quality and lending risk.
- Comparing funded amount and received amount provides a clearer picture of repayment performance.
- Segmentation by state, loan purpose, term, and borrower profile reveals deeper lending patterns.
- Monthly trends support monitoring of lending growth and portfolio changes over time.
